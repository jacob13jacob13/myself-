其實 JavaScript 的重點就是怎麼操作 DOM 以及跟後端 server 溝通，所以這週的重點也滿明確的。
先隨意從 JavaScript 的基本語法示範，再來就示範怎麼樣拿到 DOM 物件以及執行順序的問題，解釋了為什麼會看到很多人都把 script 放在 body 最下面，是為了避免 render blocking 的問題，因為瀏覽器一碰到 script 就會先停下來執行完畢。
